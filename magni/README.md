# Magni

**Magni** - React Native (Expo) app with web support via expo-router.

Frontend application for weight tracking, workout planning, and macro tracking.

## Setup

Install dependencies:
```bash
pnpm install
```

## Development Commands

- **Start dev server**: `pnpm start`
- **Run on web**: `pnpm run web`
- **Run on Android**: `pnpm run android`
- **Run on iOS**: `pnpm run ios`

## Testing Commands

- **Run tests**: `pnpm test`
- **Watch mode**: `pnpm test:watch`
- **Coverage**: `pnpm test:coverage`

## Code Quality

- **Lint**: `pnpm lint`
- **Fix linting issues**: `pnpm lint:fix`
